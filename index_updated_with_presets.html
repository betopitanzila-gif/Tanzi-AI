Direct Debit forms now auto-tick “Set-up a new plan” and “One-off contribution”, split the 6-digit BSB into the separated BSB boxes, and prevent duplicate overlay text on fillable bank fields.

    Overlay mapping for flat PDFs

    Use this when the PDF has no fillable fields. Coordinates use PDF points from the bottom-left. The Macquarie Pension Update preset is included.

    Move text option: after loading a preset, use the X/Y boxes below to move each text. Increase X = right, decrease X = left, increase Y = up, decrease Y = down. Then generate again.

    Rules for this form

    Use this to save instructions for each stored form or currently uploaded form. Examples: ignore Section 2A Rollover from another fund, write N/A in Section 3A, or write custom text in a nominated spot. These rules are saved in this browser against the form name.

    Note: “Ignore” means the dashboard will not write anything to that section. “Write N/A” and “Custom text” use overlay coordinates, so you can place the text exactly where the form needs it.
